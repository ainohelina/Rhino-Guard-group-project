# Rhino-Guard-group-project
Group project for the course User Interface construction. All the codes and files in this project were done with Fiia-Maria (fiiamh) and the workload was divided equally. The aim of the project was to design and implement an interactive, user-oriented interface combining individual design work and collaborative development within a group. The project focused on a real-world context in which the user interface had to function in challenging conditions and support users through clear, reliable, and easy-to-learn interaction. During the project, we practiced user-centered design, justifying user interface solutions using known usability heuristics, and coordinating different perspectives into a unified user interface.
The learning objectives focused particularly on the design and prototyping of interactive user interfaces, collaboration skills in the design process, critical evaluation of design solutions, and understanding how individual components together form a coherent user interface that supports user needs.
